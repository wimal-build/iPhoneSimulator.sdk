웹 사이트에 사용자를 추적하지 말도록 요청

일부 타사업체가 사용자에게 콘텐츠를 제공할 때 탐색 활동을 추적하여 사용자에게 표시할 콘텐츠를 조정할 수 있습니다. Safari에서 사이트 및 광고업체를 포함한 타사업체의 콘텐츠 제공업체에 사용자를 추적하지 말도록 요청할 수 있습니다.

이 설정을 활성화하면 Safari가 웹 사이트에서 콘텐츠를 가져올 때마다 사용자를 추적하지 말도록 Safari가 요청하지만 이 요청을 수락하느냐는 해당 웹 사이트에 달려 있습니다.

쿠키 및 기타 웹 사이트 데이터 차단

웹 사이트는 사용자의 장비에 쿠키 및 기타 데이터를 가끔 저장합니다. 이 데이터에는 이름, 이메일 주소 및 환경설정과 같은 사용자가 제공한 정보가 포함되어 있을 수 있습니다. 이 데이터는 사용자가 해당 웹 사이트에 다시 접근할 때 사용자를 인식할 수 있도록 도와주므로 해당 사이트가 사용자에게 서비스를 제공하고 사용자 관심 기반의 정보를 표시할 수 있습니다.

기본적으로 Safari는 사용자가 방문한 웹 사이트에서만 쿠키 및 웹 사이트 데이터를 허용합니다. 이로 인해 특정 광고업체가 사용자의 장비에 데이터를 저장하는 것을 방지할 수 있습니다. 환경설정을 변경하여 Safari가 쿠키 및 기타 웹 사이트 데이터를 항상 허용하거나 항상 차단하도록 할 수 있습니다.

스마트 검색 필드

Safari에는 검색 및 웹 주소를 위한 한 개의 필드가 있으며 이 편리한 필드에서 웹을 탐색할 수 있습니다. 검색 엔진 제안이 활성화되면 Safari는 사용자가 입력한 내용을 기반으로 하는 제안에 대해 선택한 검색 엔진을 요청합니다.

많이 방문한 사이트 미리 로드가 활성화되어 있는 경우 Safari가 사용자의 책갈피 및 탐색 기록을 기반으로 하여 많이 방문한 사이트를 결정하는 대로 Safari가 백그라운드에 웹 페이지를 로드하기 시작합니다. 이 옵션을 비활성화하는 경우 해당 페이지는 정상적으로 로드됩니다.

위조된 웹 사이트 경고

위조된 웹 사이트 경고를 활성화하면 방문한 사이트가 의심되는 피싱 사이트인 경우 Safari에서 경고가 표시됩니다. 피싱은 사용자 이름, 암호 및 기타 계정 정보와 같은 개인 정보를 잘못된 방식으로 도용하려고 하는 것입니다. 위조된 웹 사이트는 은행, 금융 기관 또는 이메일 서비스 제공업체와 같은 적법한 사이트로 가장합니다.

개인정보 보호 브라우징

개인정보 보호 브라우징을 활성화하면 Safari가 방문한 페이지, 자동 완성 정보 및 iCloud에 저장하지 않은 열려 있는 탭은 기억하지 않습니다. Safari는 사이트 및 타사 콘텐츠 제공업체(광고업체 포함)에도 사용자를 추적하지 않도록 요청합니다. 웹 사이트는 장비에 저장된 정보를 수정할 수 없으므로 이러한 사이트에서 사용하는 서비스는 일반적으로 개인정보 보호 브라우징을 끌 때까지 다르게 동작될 수 있습니다.

참고: 개인정보 보호 브라우징이 활성화되어 있는 동안 Safari 사용자 인터페이스가 밝게 나타나지 않고 어둡게 나타납니다.
