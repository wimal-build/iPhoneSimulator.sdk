Αίτημα σε ιστότοπους για τη διακοπή ανίχνευσης

Κάποιοι ιστότοποι ανιχνεύουν τις δραστηριότητες περιήγησής σας όταν σας στέλνουν περιεχόμενο, κάτι που τους δίνει τη δυνατότητα να προσαρμόζουν αυτά που σας εμφανίζουν. Το Safari μπορεί να ζητήσει από ιστότοπους, και τους τρίτους πάροχους περιεχομένου τους (συμπεριλαμβανομένων διαφημιστών), να μην ανιχνεύουν τη δραστηριότητά σας.

Όταν είναι ενεργοποιημένη αυτή η ρύθμιση, κάθε φορά που το Safari λαμβάνει περιεχόμενο από έναν ιστότοπο, προσθέτει ένα αίτημα να μη σας ανιχνεύει ο ιστότοπος αλλά εξαρτάται από τον συγκεκριμένο ιστότοπο αν θα ικανοποιηθεί το αίτημά σας ή όχι.

Αποκλεισμός cookie και άλλων δεδομένων ιστότοπων

Οι ιστότοποι συχνά αποθηκεύουν cookie και άλλα δεδομένα στη συσκευή σας. Αυτά τα δεδομένα μπορεί να περιλαμβάνουν πληροφορίες που έχετε εισαγάγει, όπως π.χ. το όνομα, τη διεύθυνση email και τις προτιμήσεις σας. Αυτά τα δεδομένα βοηθούν τους ιστότοπους να σας ταυτοποιούν μετά την πρώτη επίσκεψή σας ώστε να μπορεί ο εκάστοτε ιστότοπος να σας παρέχει υπηρεσίες και να σας εμφανίζει πληροφορίες που μπορεί να σας ενδιαφέρουν.

Από προεπιλογή, το Safari αποδέχεται cookie και δεδομένα ιστότοπων μόνο από ιστότοπους που επισκέπτεστε. Αυτό δεν επιτρέπει σε ορισμένους διαφημιστές την αποθήκευση δεδομένων στη συσκευή σας. Μπορείτε να αλλάξετε τις προτιμήσεις σας ώστε το Safari να αποκλείει ή να αποδέχεται πάντα όλα τα cookie και άλλα δεδομένα ιστότοπων.

Προειδοποίηση δόλιου ιστότοπου

Όταν είναι ενεργοποιημένη η Προειδοποίηση δόλιου ιστότοπου, το Safari θα εμφανίσει μια προειδοποίηση ότι ο ιστότοπος που επισκέπτεστε είναι ύποπτος ιστότοπος «ψαρέματος». Το «ψάρεμα» είναι μια δόλια απόπειρα κλοπής των προσωπικών σας δεδομένων, όπως π.χ. ονόματα χρήστη, συνθηματικά και άλλες πληροφορίες λογαριασμών. Ένας δόλιος ιστότοπος μεταμφιέζεται ως νόμιμος ιστότοπος, όπως π.χ. τράπεζα, χρηματοπιστωτικός οργανισμός ή πάροχος υπηρεσιών email.

Ιδιωτική περιήγηση

Όταν είναι ενεργοποιημένη η Ιδιωτική περιήγηση, το Safari δεν απομνημονεύει τις σελίδες που επισκέπτεστε και τις πληροφορίες Αυτοσυμπλήρωσης και οι ανοικτές καρτέλες σας δεν αποθηκεύονται στο iCloud. Οι ιστότοποι δεν μπορούν να τροποποιήσουν πληροφορίες που είναι αποθηκευμένες στη συσκευή σας, συνεπώς οι υπηρεσίες που διατίθενται συνήθως σε τέτοιους ιστότοπους μπορεί να λειτουργούν διαφορετικά έως ότου απενεργοποιήσετε την Ιδιωτική περιήγηση.

Σημείωση: Ενώ είναι ενεργοποιημένη η Ιδιωτική περιήγηση, το περιβάλλον χρήστη στο Safari εμφανίζεται σκούρο αντί για ανοιχτόχρωμο.

Προτάσεις Spotlight

Όταν χρησιμοποιείτε τις Προτάσεις Spotlight στο Safari, τα ερωτήματα αναζήτησής σας, οι Προτάσεις Spotlight που επιλέγετε και σχετικά δεδομένα αναζήτησης θα αποστέλλονται στην Apple. Αν έχετε ενεργοποιήσει τις Υπηρεσίες τοποθεσίας στη συσκευή σας, όταν κάνετε ένα ερώτημα αναζήτησης στο Spotlight, η τοποθεσία της συσκευής σας τη δεδομένη στιγμή θα αποστέλλεται στην Apple. Η τοποθεσία, τα ερωτήματα αναζήτησης και οι πληροφορίες χρήσης που στέλονται στην Apple θα χρησιμοποιούνται από την Apple μόνο για να γίνουν οι Προτάσεις Spotlight πιο σχετικές και για τη βελτίωση άλλων προϊόντων και υπηρεσιών Apple.

Αν δεν επιθυμείτε την αποστολή στην Apple των ερωτημάτων αναζήτησης στο Safari και των δεδομένων χρήσης των Προτάσεων Spotlight, μπορείτε να αλλάξετε τις ρυθμίσεις απορρήτου σας για το Safari ανά πάσα στιγμή. Για να το κάνετε αυτό, ανοίξτε τις «Ρυθμίσεις», αγγίξτε Safari, κάντε κύλιση προς τα κάτω για να βρείτε το «Απόρρητο και ασφάλεια» και μετά επιλέξτε τις ρυθμίσεις που θέλετε να ενεργοποιήσετε ή να απενεργοποιήσετε.

Μπορείτε να απενεργοποιήσετε τις Υπηρεσίες τοποθεσίας για τις Προτάσεις Spotlight οποιαδήποτε στιγμή. Για να το κάνετε αυτό, ανοίξτε τις «Ρυθμίσεις», αγγίξτε «Απόρρητο», «Υπηρεσίες τοποθεσίας» και σύρετε τον διακόπτη των Προτάσεων Spotlight στο «Όχι». Αν απενεργοποιήσετε τις Υπηρεσίες τοποθεσίας στη συσκευή σας, η ακριβής τοποθεσία σας δεν θα αποστέλλεται στην Apple. Για την παροχή σχετικών προτάσεων αναζήτησης, η Apple μπορεί να χρησιμοποιεί τη διεύθυνση IP της σύνδεσής σας στο Διαδίκτυο για την εκτίμηση κατά προσέγγιση της τοποθεσίας σας αντιστοιχίζοντας αυτήν τη διεύθυνση με μια γεωγραφική περιοχή.

Ο χειρισμός των πληροφοριών που συλλέγει η Apple γίνεται σύμφωνα με την Πολιτική απορρήτου της Apple, την οποία μπορείτε να βρείτε στη διεύθυνση www.apple.com/gr/privacy.
