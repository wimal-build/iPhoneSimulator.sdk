Meminta tapak web untuk tidak menjejak anda

Sesetengah tapak web menjejak aktiviti pelayaran anda apabila mereka menyajikan anda kandungan, yang membolehkan mereka memperibadikan kandungan yang mereka persembahkan kepada anda. Safari boleh meminta tapak web dan pembekal kandungan pihak ketiga mereka (termasuk pengiklan) untuk tidak menjejak anda.

Dengan seting ini didayakan, setiap kali Safari mengambil kandungan daripada tapak web, Safari menambah permintaan untuk tidak menjejak anda, tetapi terpulang kepada tapak web untuk menunaikan permintaan ini.

Sekat kuki dan data tapak web yang lain

Tapak web seringkali menyimpan kuki dan data lain pada peranti anda. Data ini mungkin termasuk maklumat yang telah anda beri, seperti nama, alamat e-mel dan keutamaan anda. Data ini membantu tapak web untuk mengenal pasti anda apabila anda kembali agar tapak web boleh memberi anda perkhidmatan dan menunjukkan maklumat yang mungkin anda minati.

Secara lalai, Safari hanya menerima kuki dan data tapak web daripada tapak web yang anda lawati. Ini membantu menghalang pengiklan tertentu daripada menyimpan data pada peranti anda. Anda boleh menukar keutamaan anda agar Safari sentiasa menerima atau sentiasa menyekat kuki dan data tapak web lain.

Medan Carian Lalai

Safari mempunyai satu medan untuk carian dan alamat web agar anda boleh melayari web dari satu tempat yang mudah. Apabila Cadangan Enjin Carian didayakan, Safari akan meminta enjin carian yang anda pilih untuk cadangan berdasarkan apa yang anda taipkan.

Dengan Pramuat Hasil Teratas didayakan, sebaik sahaja Safari menentukan Hasil Teratas berdasarkan penanda buku dan sejarah pelayaran anda, Safari akan mula memuatkan halaman web di latar belakang. Jika anda menyahdayakan pilihan ini, halaman akan dimuatkan seperti biasa.

Amaran Tapak Web Penipuan

Apabila Amaran Tapak Web Penipuan didayakan, Safari akan memaparkan amaran jika tapak yang anda lawati disyaki tapak memancing maklumat. Memancing ialah percubaan berbentuk penipuan untuk mencuri maklumat peribadi anda, seperti nama pengguna, kata laluan dan maklumat akaun lain. Tapak web penipuan menyamar sebagai tapak web yang sah, seperti bank, institusi kewangan atau pembekal khidmat e-mel.

Pelayaran Peribadi

Apabila Pelayaran Peribadi didayakan, Safari tidak mengingati halaman yang anda lawati, maklumat Auto Isi dan tab terbuka anda tidak disimpan dalam iCloud. Safari juga akan meminta tapak dan pembekal kandungan pihak ketiga mereka (termasuk pengiklan) untuk tidak menjejak anda. Tapak web tidak boleh mengubah suai maklumat yang disimpan pada peranti anda, maka perkhidmatan yang biasanya tersedia di tapak sedemikian mungkin berfungsi secara berlainan sehingga anda menyahaktifkan Pelayaran Peribadi.

Nota: Semasa Pelayaran Peribadi didayakan, antara muka pengguna Safari kelihatan gelap daripada cerah.
