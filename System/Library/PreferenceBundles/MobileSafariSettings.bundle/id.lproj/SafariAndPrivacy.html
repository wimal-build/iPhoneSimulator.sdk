Meminta situs web untuk tidak melacak Anda

Beberapa situs web melacak aktivitas penelusuran Anda saat Anda membuka konten situs tersebut, yang memungkinkan situs untuk mengatur apa yang ditampilkan kepada Anda. Safari dapat meminta situs, dan penyedia konten pihak ketiganya (termasuk pengiklan), untuk tidak melacak Anda.

Jika pengaturan ini diaktifkan, setiap kali Safari mengambil konten dari situs web, Safari akan menambahkan permintaan untuk tidak melacak Anda tetapi disetujui atau tidaknya permintaan tersebut merupakan keputusan situs web.

Memblokir cookie dan data situs web lainnya

Situs web seringkali menyimpan cookie dan data lainnya di perangkat Anda. Data ini dapat berisi informasi yang telah Anda berikan, seperti nama, alamat email, dan preferensi Anda. Data ini membantu situs web mengidentifikasi Anda saat Anda kembali sehingga situs dapat menyediakan layanan untuk Anda dan memperlihatkan informasi yang mungkin menarik bagi Anda.

Secara default, Safari hanya menerima cookie dan data situs web lainnya dari situs web yang Anda kunjungi. Ini membantu mencegah pengiklan tertentu untuk menyimpan data di perangkat Anda. Anda dapat mengubah preferensi Anda supaya Safari selalu menerima atau memblokir cookie dan data situs web lainnya.

Bidang Pencarian Cerdas

Safari memiliki satu bidang untuk pencarian dan alamat web sehingga Anda dapat menelusuri web di satu tempat dengan praktis. Jika Saran Mesin Pencari diaktifkan, Safari akan meminta saran kepada mesin pencari yang Anda pilih berdasarkan apa yang Anda ketik.

Jika Pramuat Hasil Teratas diaktifkan, segera setelah Safari menentukan Hasil Teratas berdasarkan penanda dan riwayat penelusuran Anda, Safari akan mulai memuat halaman web di latar belakang. Jika Anda menonaktifkan pilihan ini, halaman akan dimuat seperti biasa.

Peringatan Situs Web Palsu

Jika Peringatan Situs Web Palsu diaktifkan, Safari akan menampilkan peringatan jika situs yang Anda kunjungi dicurigai sebagai situs phishing. Phishing merupakan tindak penipuan untuk mencuri informasi pribadi Anda, seperti nama pengguna, kata sandi, dan informasi akun lainnya. Situs web palsu menyamar sebagai situs resmi, seperti bank, lembaga keuangan atau penyedia layanan email.

Penelusuran Pribadi

Jika Penelusuran Pribadi diaktifkan, Safari tidak akan mengingat halaman yang Anda kunjungi, informasi Isi-Auto, dan tab Anda yang terbuka tidak akan disimpan di iCloud. Situs web dapat memodifikasi informasi yang disimpan di perangkat Anda, sehingga layanan yang biasanya tersedia di situs tersebut bisa saja berbeda sampai Anda mematikan Penelusuran Pribadi.

Catatan: Selama Penelusuran Pribadi diaktifkan, antarmuka pengguna Safari akan tampak gelap bukan terang.
