Meminta situs web untuk tidak melacak Anda

Beberapa situs web melacak kegiatan penelusuran Anda saat mereka memberikan konten, yang memungkinkan mereka menyesuaikan apa yang mereka berikan kepada Anda. Safari dapat meminta situs dan penyedia layanan pihak ketiga mereka (termasuk pengiklan) untuk tidak melacak Anda.

Dengan mengaktifkan pengaturan ini, tiap kali Safari mengambil konten dari situs web, Safari menambahkan permintaan agar Anda tidak dilacak tapi hal tersebut tergantung pada situs web untuk menghormati permintaan tersebut atau tidak.

Memblokir coookie dan data situs web lainnya

Situs web sering menyimpan cookie dan data lain di perangkat Anda. Data tersebut dapat termasuk informasi yang telah Anda berikan, seperti nama, alamat email, dan preferensi. Data ini membantu situs web mengidentifikasi Anda saat Anda kembali, sehingga situs dapat memberikan layanan untuk Anda dan menampilkan informasi yang mungkin menarik bagi Anda.

Secara default, Safari menerima cookie dan data situs web hanya dari situs web yang Anda kunjungi. Hal ini membantu mencegah pengiklan tertentu menyimpan data di perangkat Anda. Anda dapat mengubah preferensi Anda agar Safari selalu menerima atau selalu memblokir cookie dan data situs web lainnya.

Peringatan Situs Web Palsu

Saat peringatan situs web palsu diaktifkan, Safari akan menampilkan peringatan jika situs yang Anda kunjungi diduga situs phishing. Phishing adalah usaha pemalsuan untuk mencuri informasi personal Anda, seperti nama pengguna, kata sandi, dan informasi akun lainnya. Situs web palsu menyamar sebagai situs web resmi, seperti lembaga finansial atau penyedia layanan email.

Penelusuran Pribadi

Saat penelusuran pribadi diaktifkan, Safari tidak mengingat halaman yang Anda kunjungi, informasi Isi-Auto, dan tab terbuka Anda tidak disimpan di iCloud. Safari juga akan meminta situs dan penyedia konten pihak ketiga (termasuk pengiklan) untuk tidak melacak Anda. Situs web tidak dapat memodifikasi informasi yang disimpan di perangkat Anda, jadi layanan yang biasanya tersedia di situs mungkin bekerja secara berbeda hingga Anda mematikan Penelusuran Pribadi.

Catatan: Saat Penelusuran Pribadi diaktifkan, antarmuka pengguna Safari muncul berwarna gelap, bukan terang.

Saran Spotlight

Saat Anda menggunakan Saran Spotlight di Safari, daftar pencarian Anda, Saran Spotlight yang Anda pilih, dan data penggunaan terkait akan dikirimkan ke Apple. Jika Anda menyalakan Layanan Lokasi di perangkat Anda, maka lokasi perangkat saat Anda membuat daftar permintaan ke Spotlight akan dikirimkan ke Apple. Lokasi, daftar pencarian, dan informasi penggunaan yang dikirim ke Apple akan digunakan hanya oleh Apple untuk membuat Saran Spotlight menjadi lebih relevan dan untuk meningkatkan produk dan layanan Apple lainnya.

Jika Anda tidak ingin daftar pancarian Safari dan data penggunaan Saran Spotlight Anda dikirim ke Apple, Anda dapat mengubah pengaturan privasi Safari kapan saja. Untuk melakukannya, buka Pengaturan, ketuk Safari, gulir ke bawah ke Privasi & dan Keamanan, lalu pilih pilihan Yang ingin Anda aktifkan atau nonaktifkan.

Anda dapat mematikan Layanan Lokasi untuk Saran Spotlight. Untuk melakukannya, buka Pengaturan, ketuk Privasi, dan geser sakelar Saran Spotlight menjadi “mati”. Jika Anda mematikan Layanan Lokasi di perangkat Anda, maka lokasi Anda yang tepat tidak akan dikirimkan ke Apple. Untuk memberikan saran pencarian yang relevan, Apple mungkin menggunakan alamat IP koneksi Internet Anda untuk memperkirakan lokasi Anda dengan mencocokkannya ke wilayah geografis.

Informasi yang dikumpulkan oleh Apple akan diperlakukan sesuai dengan Kebijakan Privasi Apple yang dapat ditemukan di www.apple.com/privacy.
