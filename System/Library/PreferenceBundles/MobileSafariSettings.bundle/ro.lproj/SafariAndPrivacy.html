Solicitarea neurmăririi dvs. de către site-urile web

Unele site-uri web țin evidența activităților dvs. de navigare atunci când vă servesc conținut, ceea ce le permite să adapteze ceea ce vă prezintă. Safari le poate solicita site-urilor și terților furnizori de conținut ai acestora (inclusiv furnizorilor de publicitate) să nu vă urmărească.

Dacă această configurare este activată, de fiecare dată când Safari accesează conținut de pe un site web, Safari adaugă o solicitare de a nu vă urmări, însă decizia de a respecta această solicitare aparține site-ului web.

Blocarea cookie-urilor și altor date ale site-urilor web

Site-urile web stochează frecvent cookie-uri și alte date pe dispozitivul dvs. Aceste date pot include informații furnizate de dvs, precum numele, adresa de e-mail și preferințele dvs. Aceste date ajută site-urile web să vă identifice atunci când reveniți, astfel încât site-ul să poată să vă furnizeze servicii și să vă prezinte informații care ar putea fi de interes pentru dvs.

Implicit, Safari acceptă cookie-uri și date ale site-urilor web doar din site-urile web pe care le vizitați. Aceasta ajută la prevenirea stocării datelor de către anumiți furnizori de publicitate pe dispozitivul dvs. Puteți schimba preferințele dvs. astfel încât Safari să accepte întotdeauna sau să blocheze întotdeauna cookie-urile și datele site-urilor web.

Câmpul inteligent de căutare

Safari are un câmp pentru căutări și adrese web astfel încât să puteți naviga pe web dintr-un singur loc convenabil. Când opțiunea “Sugestii motor de căutare” este activată, Safari va solicita motorului de căutare selectat sugestii bazate pe ceea ce ați tastat.

Cu preîncărcarea primului rezultat activată, imediat ce Safari determină un prim rezultat pe baza favoritelor și al istoricului dvs. de navigare, Safari va începe încărcarea paginii web în fundal. Dacă dezactivați această opțiune, pagina se va încărca normal.

Avertismentele referitoare la site-urile frauduloase

Dacă opțiunea “Avertisment site web fraudulos” este activată, Safari va afișa un avertisment dacă site-ul pe care îl vizitați este un site suspectat de “phishing”. Phishing-ul este o tentativă frauduloasă de furt al informațiilor dvs. personale, precum numele de utilizator, parolele și alte informații de cont. Un site web fraudulos imită unul legitim, precum site-ul unei banci, al unei instituții financiare sau al unui furnizor de servicii de e-mail.

Navigarea privată

Dacă opțiunea “Navigare privată” este activată, Safari nu reține paginile pe care le vizitați și informațiile de Auto-completare, iar filele dvs. deschise nu sunt stocate pe iCloud. De asemenea, Safari va solicita site-urilor și terților furnizori de conținut ai acestora (inclusiv furnizorilor de publicitate) să nu vă urmărească. Site-urile web nu pot modifica informațiile stocate pe dispozitivul dvs, astfel încât este posibil ca serviciile disponibile în mod normal pe aceste site-uri să funcționeze diferit până când dezactivați opțiunea “Navigare privată”.

Notă: În timp ce opțiunea “Navigare privată” este activată, interfața de utilizare Safari are un ton întunecat, în locul unui ton deschis.
